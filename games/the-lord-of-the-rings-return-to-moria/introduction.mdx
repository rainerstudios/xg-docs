## Self-hosting a The Lord of the Rings: Return to Moria dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD).

### System requirements

A The Lord of the Rings: Return to Moria server requires at least **8GB RAM**, **4 CPU cores**, and **20GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./rtm_server/ +login anonymous +app_update 3349480 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\rtm_server\ +login anonymous +app_update 3349480 +quit`

### Starting game server

Run the server by launching the appropriate executable with your desired parameters.
