## Self-hosting an Unturned dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Unturned server requires at least **2GB RAM**, **2 CPU cores**, and **2GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./unturned_server/ +login anonymous +app_update 1110390 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\unturned_server\ +login anonymous +app_update 1110390 +quit`

### Starting game server

Run the server by launching the Unturned executable with appropriate server parameters.
