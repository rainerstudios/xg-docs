## Self-hosting a Factorio dedicated server

You can also alternatively run a server on your own computer/PC by downloading the dedicated server from the official website.

### System requirements

A single Factorio server requires at least **2GB RAM**, **2 CPU cores**, and **1GB of storage**.

### Installing game server

Download the dedicated server from the [official Factorio website](https://factorio.com/download) and extract it to your desired location.

### Starting game server

Run the server by launching the factorio executable with appropriate parameters.
