## Self-hosting a Sons of the Forest dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Sons of the Forest server requires at least **8GB RAM**, **4 CPU cores**, and **10GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./sonsoftheforest_server/ +login anonymous +app_update 2465200 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\sonsoftheforest_server\ +login anonymous +app_update 2465200 +quit`

### Starting game server

Run the server by launching the SonsOfTheForestDS executable with appropriate parameters.
