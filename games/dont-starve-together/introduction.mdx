## Self-hosting a Don't Starve Together dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Don't Starve Together server requires at least **1GB RAM**, **1 CPU core**, and **1GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./dst_server/ +login anonymous +app_update 343050 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\dst_server\ +login anonymous +app_update 343050 +quit`

### Starting game server

Run the server by launching the dontstarve_dedicated_server_nullrenderer executable with appropriate parameters.
