## Self-hosting a Rust dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Rust server requires at least **8GB RAM**, **4 CPU cores**, and **20GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./rust_server/ +login anonymous +app_update 258550 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\rust_server\ +login anonymous +app_update 258550 +quit`

### Starting game server

Run the server by launching the RustDedicated executable with appropriate parameters.
