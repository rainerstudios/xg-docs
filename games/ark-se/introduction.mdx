## Self-hosting an ARK dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single ARK server requires at least **8GB RAM**, **4 CPU cores**, and **25GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./ark_server/ +login anonymous +app_update 376030 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\ark_server\ +login anonymous +app_update 376030 +quit`

### Starting game server

Run the server by launching the ShooterGameServer executable with appropriate parameters.
