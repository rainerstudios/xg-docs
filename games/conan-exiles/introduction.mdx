## Self-hosting a Conan Exiles dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Conan Exiles server requires at least **8GB RAM**, **4 CPU cores**, and **35GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./conanexiles_server/ +login anonymous +app_update 443030 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\conanexiles_server\ +login anonymous +app_update 443030 +quit`

### Starting game server

Run the server by launching the ConanSandboxServer executable with appropriate parameters.
