## Self-hosting a The Front dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD).

### System requirements

A The Front server requires at least **16GB RAM**, **4 CPU cores**, and **30GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./the-front_server/ +login anonymous +app_update 2334200 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\the-front_server\ +login anonymous +app_update 2612550 +quit`

### Starting game server

Run the server by launching the appropriate executable with your desired parameters.
