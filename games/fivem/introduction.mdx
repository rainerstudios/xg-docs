## Self-hosting a FiveM dedicated server

You can also alternatively run a server on your own computer/PC by downloading the FiveM server files.

### System requirements

A single FiveM server requires at least **4GB RAM**, **4 CPU cores**, and **10GB of storage**.

### Installing game server

Download the FiveM server files from the [official FiveM website](https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/) and extract them to your desired location.

### Starting game server

Run the server by launching the FXServer executable with appropriate configuration.
