## Self-hosting a Core Keeper dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Core Keeper server requires at least **4GB RAM**, **2 CPU cores**, and **5GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./corekeeper_server/ +login anonymous +app_update 1963720 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\corekeeper_server\ +login anonymous +app_update 1963720 +quit`

### Starting game server

Run the server by launching the CoreKeeperServer executable with appropriate parameters.
