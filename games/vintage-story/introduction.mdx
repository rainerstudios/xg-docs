## Self-hosting a Vintage Story dedicated server

You can also alternatively run a server on your own computer/PC by downloading the dedicated server from the official website.

### System requirements

A single Vintage Story server requires at least **4GB RAM**, **2 CPU cores**, and **3GB of storage**.

### Installing game server

Download the dedicated server from the [official Vintage Story website](https://www.vintagestory.at/download/) and extract it to your desired location.

### Starting game server

Run the server by launching the VintagestoryServer executable with appropriate parameters.
