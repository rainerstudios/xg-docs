## Self-hosting an Assetto Corsa dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD).

### System requirements

An Assetto Corsa server requires at least **4GB RAM**, **2 CPU cores**, and **10GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./assetto-corsa_server/ +login anonymous +app_update 302550 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\assetto-corsa_server\ +login anonymous +app_update 302550 +quit`

### Starting game server

Run the server by launching the appropriate executable with your desired parameters.
