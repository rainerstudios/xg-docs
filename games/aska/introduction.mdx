## Self-hosting an Aska dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD).

### System requirements

An Aska server requires at least **16GB RAM**, **4 CPU cores**, and **10GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./aska_server/ +login anonymous +app_update 3246670 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\aska_server\ +login anonymous +app_update 3246670 +quit`

### Starting game server

Run the server by launching the appropriate executable with your desired parameters.
