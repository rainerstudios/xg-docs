Want to downgrade the server version or rollback the version of your [Palworld server](https://xgamingserver.com/palworld-server-hosting)? Or update it to a specific version which isn't the latest release?

This guide explains how, we at [XGamingServer](https://xgamingserver.com), have built in a server version selector for Palworld into our [game panel](https://game.xgamingserver.com).

## Change Palworld server version

1. Login to the [game panel](https://game.xgamingserver.com) and **Stop** the server
2. Take a backup of your server (just in case) and navigate to the **Palworld Settings** tab
3. Disable **Auto Update**
<Frame>
  <img src="/images/games/palworld/auto-update-palworld.png" />
</Frame>
4. Enable **Manual Update**
<Frame>
  <img src="/images/games/palworld/manual-update-palworld.png" />
</Frame>
5. Set the manifest ID to your desired version (find the ID from [steamdb.info](https://steamdb.info/depot/2394012/manifests/))
<Frame>
  <img src="/images/games/palworld/palworld-manifest-id.png" />
</Frame>
6. Start the server!

After completing these steps, your server should be rolled back or updated to the version! For example, to roll back to Palworld version v0.3.5, input `2409259220322194279`.

Need help changing the server version of your Palworld? Shoot us a message on our
[community Discord server](https://xgamingserver.com/discord) or [contact support](https://xgamingserver.com/support).
