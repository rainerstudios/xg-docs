## Palworld Admin Commands

To use admin commands on your Palworld server, you must first set an AdminPassword in the [Palworld settings](./configure-server-settings) and become an admin on the server.

## Becoming an Admin
Once the password is set, enter the game and run the following command to gain admin privileges:

`/AdminPassword <your_admin_password>`

For a detailed guide on becoming an admin, check out [How to Become an Admin in Palworld](./become-an-admin).

## How to Execute Admin Commands
	1.	Press **Enter** in-game to open the chat.
	2.	Type the desired command and press **Enter** to execute.


## Admin Command List
| Command                       | Description                                                                                                                        |
|-------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| `/AdminPassword`                | Gain admin privileges using the set password.                                                                                      |
| `/Shutdown [Seconds] [Message]` | Shut down the server after a specified time. Players will see the message you enter. Use without [Seconds] for immediate shutdown. |
| `/DoExit`                       | Force stop the server immediately.                                                                                                 |
| `/Broadcast <Message>`         | Send a message to all players on the server.                                                                                       |
| `/KickPlayer <SteamID>`         | Kick the player with the specified Steam ID.                                                                                       |
| `/BanPlayer <SteamID>`          | Ban the player with the specified Steam ID.                                                                                        |
| `/UnBanPlayer <SteamID>`        | Unban the player with the specified Steam ID.                                                                                      |
| `/TeleportToPlayer <SteamID>`   | Teleport yourself to the player with the Steam ID.                                                                                 |
| `/TeleportToMe <SteamID>`       | Teleport the player to your location.                                                                                              |
| `/ShowPlayers`                  | Show information about all connected players.                                                                                      |
| `/Info`                         | Display server information.                                                                                                        |
| `/Save`                         | Save the current world data.                                                                                                       |

Use these commands to manage your server efficiently. If you encounter any issues, ensure you have admin rights and the correct password set.
