The new [Palworld Sakurajima update](https://blog.berrybyte.net/palworld-sakurajima-update-released/) brings [Xbox dedicated servers](https://berrybyte.net/palworld-xbox-server-hosting), and with you'll need to know how to connect and join your Palworld server on Xbox.

## Enable community server

As you can't join a Palworld server with an IP addresses you'll need to enable your server as a community server.

To configure a Palworld server as a community server, add the `-publiclobby` option to your server's startup launch arguments.

## Join and connect to Palworld Server on Xbox
1. Launch Palworld
2. Select **Join Multiplayer Game** from the main menu
<Frame>
  <img src="/images/join-palworld-server.png" />
</Frame>
3. Select **Community Servers**
4. Search for the name of your server to find it
5. Click **Connect** which will connect and load you into your multiplayer server!
