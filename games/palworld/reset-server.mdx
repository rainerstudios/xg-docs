Want to start a Palworld server from fresh, wiping the existing saves? Here's how:

## Resetting a Palworld Server

1. Login to the [game panel](https://game.xgamingserver.com)
2. In the **Files** tab, select all files, and press delete.
3. Navigate to the **Settings** tab, and press **Reinstall**

This will wipe your server and start it from a brand new fresh world!
