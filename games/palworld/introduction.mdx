## Self-hosting a Palworld dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Palworld server is recommended to have at least **16GB RAM**, **4 CPU cores**, and **20GB of storage**.

### Installing game server

**On Linux:**
`steamcmd force_install_dir ./Palworld_server/ +login anonymous app_update 2394010 quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\Palworld_server\ +login anonymous +app_update 2394010 +quit`

### Starting game server

Run the server by launching `StartServer.bat` (Windows) or `./StartServer.sh` (Linux). Make sure to have the UDP port 8211 and 27015 (query port) are port forwarded.
