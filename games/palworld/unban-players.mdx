## Unbanning Players on a Palworld Server

1. Login to the [game panel](https://game.berrybyte.net)
2. **Stop** the server
4. Navigate to **Pal** -> **Saved** -> **SaveGames** -> "**banlist.txt**"
5. Remove the SteamID of the player you wish to unban, and save the file
7. **Start** the server!
