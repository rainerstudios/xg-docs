Want to protect your server from unwanted joins? You can set a password on your Palworld server to put it behind lock and key, ensuring only you and the people you share the password with can join the server!

## Setting a password on a Palworld server

1. Login to the [game panel](https://game.xgamingserver.com)
2. Navigate to the **Palworld Settings** tab
3. Enter a password in the **Server Password** field
<Frame>
  <img src="/images/games/palworld/palworld-password.png" />
</Frame>
4. Restart the server

## Setting a password on a Palworld server (manual)

1. **Stop** the server
2. Navigate to `Pal/Saved/Config/LinuxServer/PalWorldSettings.ini`
3. Enter your password in `ServerPassword="replaceWithYourPassword"`
<Frame>
  <img src="/images/games/palworld/palworldsettings-ini-password.png" />
</Frame>
4. **Save** the file
5. **Start** the server

This will set the password you entered and will ask all players who join for the password.
