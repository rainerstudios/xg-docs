## Self-hosting a Garry's Mod dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Garry's Mod server requires at least **2GB RAM**, **2 CPU cores**, and **5GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./gmod_server/ +login anonymous +app_update 4020 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\gmod_server\ +login anonymous +app_update 4020 +quit`

### Starting game server

Run the server by launching the srcds executable with appropriate Garry's Mod parameters.
