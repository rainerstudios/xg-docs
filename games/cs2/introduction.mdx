## Self-hosting a Counter-Strike 2 dedicated server

You can also alternatively run a server on your own computer/PC with [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD) or from Steam library.

### System requirements

A single Counter-Strike 2 server requires at least **4GB RAM**, **4 CPU cores**, and **15GB of storage**.

### Installing game server

**On Linux:**
`steamcmd +force_install_dir ./cs2_server/ +login anonymous +app_update 730 +quit`

**On Windows:**
`steamcmd.exe +force_install_dir c:\SteamCMD\cs2_server\ +login anonymous +app_update 730 +quit`

### Starting game server

Run the server by launching the cs2 executable with appropriate server parameters.
